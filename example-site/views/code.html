Dragonfly Source Code

Dragonfly is hosted on Google Code. You got two options for getting Dragonfly up and running:

No 1. Get the latest, bleeding edge version (Mercurial repository)

  • Make sure You're able to run the hg command (Mercurial)
  • Open Your shell
  • Enter this command
  • hg clone https://dragonfly-newlisp.googlecode.com/hg/ dragonfly-newlisp

No 2. Get the latest stable version

  • Open Your browser and download the ZIP archive from this address
  • Latest stable version
