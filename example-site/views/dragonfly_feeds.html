Reading feeds from other websites

Dragonfly provides some simple functions for reading atom feeds (RSS will follow). The name Atom applies to a pair of related standards. The Atom Syndication Format is an XML language used for web feeds, while the Atom Publishing Protocol (AtomPub or APP) is a simple HTTP-based protocol for creating and updating web resources. (source: wikipedia.org)

How to load an atom feed

Example: open your view and enter the following:

<% (read-atom-feed "http://www.example-site.com/atomfeed.xml") >%

INFORMATION
If you want to see the raw XML, then just put ", true" after the feed-URL. See the example below.

<% (read-atom-feed "http://www.example-site.com/atomfeed.xml", true) >%

How to load an RSS feed

Example: open your view and enter the following:

<% (read-rss-feed "http://www.example-site.com/rssfeed.xml") >%

INFORMATION
If you want to see the raw XML, then just put ", true" after the feed-URL. See the example below.

<% (read-rss-feed "http://www.example-site.com/rssfeed.xml", true) >%
